# 🎯 機構級：宏觀與生命週期資產配置系統 (PRO版)

由 Python Streamlit 建構的全自動化視覺投資儀表板。系統自動介接聯準會 (FRED) 與 Yahoo Finance 的關鍵宏觀經濟指標，並納入生命週期資產配置邏輯的精確計算，為現代投資人打造對沖基金級別的投資防護網。

## ✨ 核心特色 (三層作戰系統)

### 1. 底層資產模型 (生命週期及滑動路徑)
- 根據使用者的 `目前年齡` 與 `預計退休年齡` 動態打造符合風險承受度的基礎資產配置。
- **累積期**：追求極致複利，最高提供 100% 股權曝險。
- **轉換期 (Glide Path)**：退休前倒數 10 年自動啟動財富保護傘，股票部位每年按比率逐步轉為防禦資產。
- **提款期**：進入退休狀態後，自動鎖定最大化資產生存率的「60/30/10 提款期模型」。

### 2. 頂層動態微調 (4D 宏觀景氣調整)
- 透過自動抓取最新的 **CPI (通膨)**、**聯邦基準利率方向**、**10Y-2Y 殖利率倒掛差**，以及 **ISM 採購經理人指數 (PMI)**，自動化判定當前景氣：
  包含 **復甦 (Recovery) / 過熱 (Overheat) / 滯脹 (Stagflation) / 衰退 (Recession)**。
- 景氣循環的變化將自動反映在組合微調之中 (自動增減，且嚴格將波動鎖定在 ±15% 以內防止過度猜測市場)。

### 3. 終極雙重防護網 (核心賣點)
- **Neutral 不動機制**：當通膨穩健停留在 2%~3% 的黃金區塊時，只要市場多空陷入拉扯，系統會強制關閉擇時微調並祭出「中性防護」，避免因震盪而導致的無意義過度交易。
- **雙極端災害防護**：只有當景氣階段落入「衰退」的同時，還發生了「10Y-2Y 殖利率實質倒掛」，系統才會真正拉響紅色警報，一次性重度防禦。

## 📊 視覺化互動油表 (Gauge Charts)

內建基於 `Plotly` 的 4D 動態儀表板，將艱澀的數值全部化作綠、黃、紅的直覺提示區間：
- **CPI 通膨 YoY**: 🟢 `<2` (舒適) / 🟡 `2-3` (中立模糊) / 🔴 `>3` (過熱警報)
- **聯準會基準利率**: 🟢 `<2` (寬鬆) / 🟡 `2-4` (中性利率) / 🔴 `>4` (緊縮壓抑)
- **利差 10Y-2Y**: 🔴 `<0` (倒掛衰退警報) / 🟡 `0-1` (平緩過渡) / 🟢 `>1` (牛市陡峭)
- **VIX 市場恐慌恐懼**: 🔵 `<15` (過度樂觀警惕) / 🟢 `15-20` (平穩) / 🟡 `20-30` (恐慌上升) / 🔴 `>30` (極度恐慌尋找買點)

## 💻 安裝與本地端運行 (Run Locally)

這是一個標準的 Python Streamlit 專案，確保您已安裝 Python 3.9+ 的環境後：

```bash
# 1. 安裝必要依賴套件
pip install -r requirements.txt

# 2. 啟動本機伺服器
streamlit run macro_dashboard.py
```
執行後請至 `localhost:8501` 查看儀表板。
